✔️React Hooks

✔️PostgreSQL

✔️Node

✔️Axios


## Projeto

GoBarber é uma aplicação com o objetivo de agendar cortes de cabelo/barba conforme a agenda do barbeiro.

## Configuração

1- Para instalar as dependências:
> yarn

2- Para iniciar a aplicação:
> yarn start
